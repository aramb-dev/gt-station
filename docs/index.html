Dashboard Modules

Six Panels. Zero Tab Switching.

Everything you need to oversee Gas Town in one native app.

01

✉︎ Mail

Thread-grouped inbox with compose, reply, priority levels, and native macOS notifications.

02

📡 Nudge

iMessage-style agent messaging with immediate, scheduled, and custom delivery modes.

03

⚙︎ Rigs

Full lifecycle management. Start, stop, dock, and undock infrastructure from one panel.

04

🗄 Dolt Health

Direct MySQLNIO monitoring with real-time health metrics and database status checks.

05

📋 Contacts

Agent directory with quick-action nudge and mail capabilities from the contacts view.

06

⚠︎ Escalations

Centralized critical alerts. Stay ahead of issues before they spread across town.
